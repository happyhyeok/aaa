# 로또 5게임 추천기 🎰

간단하고 깔끔한 디자인의 로또 번호 추천 웹 어플리케이션입니다.

## 주요 기능
- **5게임 일괄 생성**: 한 번의 클릭으로 5개의 로또 번호 세트를 생성합니다.
- **다크/화이트 모드**: 사용자 취향에 맞는 테마 선택이 가능하며 설정이 저장됩니다.
- **제휴 문의**: Formspree를 통한 간편한 메일 문의 기능을 제공합니다.
- **댓글 시스템**: Disqus를 통한 사용자 소통 기능을 지원합니다.

## 기술 스택
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## 배포
- Cloudflare Pages를 통해 호스팅되고 있습니다.
